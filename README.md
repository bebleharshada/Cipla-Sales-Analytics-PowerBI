CIPLA Sales & Profitability Analytics Dashboard

1. Project Title / Headline
CIPLA Sales & Profitability Analytics Dashboard
A dynamic, interactive Power BI report designed to analyze pharmaceutical sales, regional performance, product profitability, therapy mix, customer segmentation, and inventory efficiency using Cipla-like commercial data.

This dashboard helps business analysts, marketing teams, and leadership gain real-time insights for strategic and operational decision-making.

2. Short Description / Purpose
The Cipla Analytics Dashboard is a comprehensive Power BI solution that delivers end-to-end commercial insights across:
Sales performance
Target achievement
Profitability
Product category behavior
Therapy area contribution
Customer-type segmentation
Inventory planning

It supports pharmaceutical decision-makers with a single source of truth to optimize sales strategy, product focus, region planning, and supply chain efficiency.

3. Tech Stack
This dashboard was built using the following tools and technologies:
Power BI Desktop – Main data visualization and report creation tool.
Power Query – Data transformation, cleaning, and merging of multiple CSV files.
DAX (Data Analysis Expressions) – For advanced measures, KPIs, and dynamic calculations.
Data Modeling – Star schema modeling of Fact (Sales, Inventory) and Dimension (Product, Region, Date) tables.

4. Data Source
Cipla-like sample dataset designed for learning and analytics projects.
The dataset includes:
Sales transactions (region, customer type, product)
Net sales, gross sales, units sold, COGS
Product master (category, therapy area, pricing)
Regional sales targets
Inventory stock levels (opening/closing stock)

The dataset is synthetic/anonymized, created for educational and portfolio purposes.

5. Features / Highlights
Many pharmaceutical companies struggle with:
Fragmented sales data
No unified system to view profitability
Limited visibility into product mix or therapy performance
Inventory challenges leading to stockouts

This Power BI dashboard solves these challenges by providing actionable insights in one place.

6. Dashboard Goal
To provide a one-stop interactive platform for analyzing Cipla’s commercial performance — enabling leadership and analysts to improve:
Sales achievement
Profitability
Category & therapy planning
Customer segmentation
Inventory management

8. Business Impact / Insights

i. Regional Strategy Optimization
Identify regions below monthly target:- Plan field-force interventions.

ii. Product Profitability
Detect high-revenue but low-margin products:- Adjust pricing/discount strategies.

iii. Therapy Area Focus
See which therapy areas are driving majority sales:- Align marketing spend.

iv. Inventory Planning
Identify products with low closing stock:- Prevent stockouts and lost sales.

v. Customer Segment Strategy
Adjust hospital vs retail sales strategy based on contribution and margin.
